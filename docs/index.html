The core workspace

See every stitch before your machine does.

OpenStitch puts conversion settings, stitch-path inspection, thread matching, design health, and export in one focused desktop workspace.

01

Convert with control

Set physical size, fill spacing, detail preservation, stitch length, fill type, and path-planning priorities.

02

Inspect the path

Zoom, pan, animate, and inspect jumps, trims, color changes, density, needle points, and machine-frame fit.

03

Plan real thread

Match design colors to Madeira or Floriani, track inventory, estimate usage, and build a practical shopping list.

04

Refine and export

Reorder or merge color blocks, edit stitches, save projects, generate presentation previews, and export PES.

Paid add-ons

Go beyond conversion.

Keep the open-source core focused, then add specialized creative tools when your workflow needs them. Licensed add-ons install from a ZIP and remain available offline.

01 Paid add-on

Full Editor

Build and reshape artwork without leaving OpenStitch. Flatten images, use installed Windows fonts, add text and geometric shapes, draw freehand, and optimize item order.

  • Image color flattening
  • Text, shapes, and freehand drawing
  • Stitch-order optimization
  • Render directly into the core workspace
02 Paid add-on

Cross-Stitch Pattern

Turn artwork into a DMC-matched chart for hand stitching or a machine-embroidered full-cross design that loads directly into the OpenStitch workspace.

  • DMC thread-color matching
  • Configurable fabric count and chart size
  • Printable PNG and PDF charts
  • Two-pass full crosses with PES output

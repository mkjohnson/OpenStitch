Embroidery digitizing utility

OpenStitch

Convert SVG, JPG, PNG, PDF, and PES designs into Brother-compatible embroidery files, inspect stitch paths, plan thread colors, and troubleshoot before the design reaches the machine.

Run Locally View Source

What it does

Desktop-first embroidery workflow

Convert artwork

Import SVG, JPG, PNG, PDF, and PES files. Generate Brother PES output with adjustable size, fill spacing, max stitch length, color flattening, and fill mode.

Inspect stitch paths

Use the animated toolpath preview, slider, zoom, pan, measurements, jumps, trims, color-change markers, and optional needle points to understand the actual stitch sequence.

Plan thread

Match design colors against known Floriani thread colors, track inventory, estimate usage, and build a shopping list for colors that need to be purchased.

Edit and export

Deselect or merge color blocks, apply changes, save projects, email project bundles, draw or delete stitches, and export a fresh PES file.

Run locally

The converter needs Python or the Windows EXE

GitHub Pages can show this project site, but it cannot run the Python backend, write PES files, or access your local embroidery project files. Use the desktop app or local web server for real conversions.

From source

pip install -r requirements.txt
python app_launcher.py

Local web UI

python app.py
# open http://127.0.0.1:8765/

Build local EXE

powershell -ExecutionPolicy Bypass -File .\build_exe.ps1

What this site is for

This static site is a public front door for OpenStitch: documentation, screenshots, supported formats, local setup, and links to the repository. The generated EXE remains local build output and should not be committed to GitHub.

OpenStitch is a work in progress. Test carefully before stitching on your machine. github.com/mkjohnson/OpenStitch

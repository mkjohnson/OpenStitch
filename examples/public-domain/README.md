# Public-Domain SVG Examples

These SVG files are used to generate sample OpenStitch screenshots for the GitHub Pages site.

- `star-15.svg`: Openclipart `star-15` by PublicDomainSymbols. Listed as CC0 1.0 / public domain.
- `retro-flower.svg`: Openclipart `Retro Flower` by Nizips. Listed with a public-domain source.
- `butterfly.svg`: Openclipart `Butterfly` by worldlabel. Listed as a public-domain image from U.S. Fish and Wildlife Service source material.

These files are examples only. They are not bundled as official machine-ready embroidery designs.
